Numerade Video API

API Documentation

Access Numerade educational videos programmatically through our API endpoints.

GET Endpoint

GET /api/numerade?url=https://www.numerade.com/questions/your-question-url

Redirects directly to the video file

POST Endpoint

POST /api/numerade Content-Type: application/json { "url": "https://www.numerade.com/questions/your-question-url" }

Returns video information in JSON format

Response Format

{ "url": "https://cdn.numerade.com/encoded/...", "title": "Question Title" }
