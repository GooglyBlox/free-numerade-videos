Numerade Video API

API Documentation

Access Numerade educational videos programmatically through our API endpoints.

POST Endpoint

POST /api/getVideoSource Content-Type: application/json { "url": "https://www.numerade.com/questions/your-question-url" }

Returns a temporary access key for the video

Response Format

{ "key": "temporary_access_key", "title": "Question Title", "proxyUrl": "/api/getVideoSource?key=temporary_access_key" }

The proxyUrl is valid for 30 minutes
